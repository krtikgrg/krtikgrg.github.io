Event Management System

A self sufficient Software Application for service providers and customers, where service providers can release information about their services and customers can compare and buy them for an event along with the feature to view all the past and ongoing events. The software also informs the user if any of the services is missing while organising an event.

Tweaking XV6

Deep Dived into xv6 OS, originally using Round Robin Scheduler. I tried to improve it by adding other Schedulers like PBS(Priority Based Scheduler), MLFQ(Multi Level Feedback Queue Scheduling) and FCFS(First Come and First Serve Scheduler). The scheduler can be specified while running the intiating command. Some more user commands like waitx and a specific command{benchmark} to compare the performance of all the schedulers were implemented

OpenGl Game| Bellman Ford Application

Developed a game using OpenGl following OOPS concepts making real time use of Bellman Ford algorithm to deduce the path for the imposter to reach out to the player. The maze for the game is randomly made and will change each time you launch the game. For rest of the features visit the github repository.
